AMBEAM

AMBEAM, NUMBEAMS
For multiple-beam printers, specifies the number of beams in an additive manufacturing analysis.

SOLUTION: Additive Manufacturing
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

NUMBEAMS

Number of beams used in the build process. Default = 1.

Command Default

One beam is used in the build process.

Notes

This command is also valid in PREP7.

For more information, including a list of the elements and commands used in an additive manufacturing analysis, see AM Process Simulation in ANSYS Workbench in the Workbench Additive Manufacturing Analysis Guide.

Menu Paths

This command cannot be accessed from a menu.
