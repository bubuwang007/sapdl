PSYS

PSYS, KCN
Sets the PML element coordinate system attribute pointer.

PREP7: Meshing
PREP7: Elements
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

KCN

Coordinate system number:

0  

 — 

Use the global Cartesian coordinate system as the PML element coordinate system (default).

N  

 — 

Set the PML element coordinate system orientation based on a local Cartesian coordinate system N (where N must be greater than 10) defined by the LOCAL or CS command (for example: LOCAL,11,0).

Command Default

The PML element coordinate system orientation defaults to the global Cartesian system.

Notes

This command identifies the local coordinate system used to define the PML (perfectly matched layers) coordinate system of subsequently defined PML elements. It is only applicable to volume elements that support PML. The use of PML coordinate systems is similar to element coordinate systems, as discussed in Understanding the Element Coordinate System. N refers to the coordinate system reference number (KCN) defined using the LOCAL (or similar) command.

Menu Paths

This command cannot be accessed from a menu.
