2.9. AUX2 Commands

These commands examine or manipulate the contents of binary files produced by the program. The commands are grouped by functionality.

Table 2.94: Binary File Dump

These AUX2 commands dump the contents of binary files.
/AUX2Enters the binary file dumping processor. 
DUMPDumps the contents of a binary file. 
FILEAUX2Specifies the binary file to be dumped. 
FORMSpecifies the format of the file dump. 
PTRDumps the record of a binary file. 

Table 2.95: Binary File Manipulation

These AUX2 commands manipulate the contents of binary files.
COMBINE

Combines distributed memory parallel (Distributed ANSYS) files.

 
HBMATWrites an assembled global matrix in Harwell-Boeing format. 
PSMATWrites an assembled global matrix to a postscript format that graphically displays nonzero matrix values.
