2.12. AUX15 Commands

These commands read in an IGES file for analysis in ANSYS. The commands are grouped by functionality.

Table 2.100: IGES

These AUX15 commands read an IGES format file into PREP7 data.
/AUX15Enters the IGES file transfer processor. 
IGESINTransfers IGES data from a file into ANSYS. 
IOPTNControls options relating to importing a model.
