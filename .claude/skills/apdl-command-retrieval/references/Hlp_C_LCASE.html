LCASE

LCASE, LCNO
Reads a load case into the database.

POST1: Load Case Calculations
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

LCNO

Load case pointer number [LCDEF,STAT]. Defaults to 1.

Command Default

Load case 1.

Notes

Reads a load case into the database. Load cases are created as described on the LCDEF or LCWRITE commands. The results portion of the database and the applied forces and displacements are cleared before reading the data in. Absolute values [LCABS] and scale factors [LCFACT] can be applied during the read operation.

Menu Paths

Main Menu>General Postproc>Load Case>Read Load Case
