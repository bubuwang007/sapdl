2.13. Mapping Processor Commands

The /MAP processor allows you to map data from an external file onto the existing geometry.

Table 2.101: Pressure Mapping

These commands read pressure data from an external file and map it onto a geometry.
FTYPESpecifies the file type and the pressure type for the subsequent import of source points and pressures. 
MAPMaps pressures from source points to target surface elements. 
/MAPEnters the mapping processor. 
PLGEOMPlots target and source geometries. 
PLMAPPlots target and source pressures. 
READReads coordinate and pressure data from a file. 
TARGETSpecifies the target nodes for mapping pressure onto surface effect elements. 
WRITEMAPWrites interpolated pressure data to a file.
