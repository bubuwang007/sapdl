BFEDELE

BFEDELE, ELEM, Lab
Deletes element body force loads.

SOLUTION: FE Body Loads
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

ELEM

Element at which body load is to be deleted. If ALL, delete for all selected elements [ A component name may also be substituted for ELEM.

Lab

Valid body load label. If ALL, use all appropriate labels. See BFE command for labels.

Notes

Deletes element body force loads for a specified element and label. Element body loads may be defined with the BFE commands.

Graphical picking is available only via the listed menu paths.

This command is also valid in PREP7.
