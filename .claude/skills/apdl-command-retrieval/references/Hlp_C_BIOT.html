BIOT

BIOT, Label
Calculates the Biot-Savart source magnetic field intensity.

SOLUTION: Misc Loads
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

Label

Controls the Biot-Savart calculation:

NEW

 — 

Calculate the magnetic source field intensity (Hs) from the selected set of source elements to the selected set of nodes. Overwrite any existing Hs field values.

SUM

 — 

Calculate the Hs field from the selected set of source elements to the selected set of nodes. Accumulate with any existing Hs field values.

Command Default

Calculate the Hs field upon encountering the first SOLVE command to produce a source field.

Notes

Calculates the Biot-Savart source magnetic field intensity (Hs) at the selected nodes from the selected source elements. The calculation is done at the time the BIOT command is issued.

Source elements include primitives described by element SOURC36, and coupled-field elements SOLID5, LINK68, and SOLID98. Current conduction elements do not have a solved-for current distribution from which to calculate a source field until after the first substep. Inclusion of a current conduction element Hs field will require a subsequent BIOT,SUM command (with SOURC36 elements unselected) and a SOLVE command.

The units of Hs are as specified by the current EMUNIT command setting.

This command is also valid in PREP7.
