FJLIST, Elem
Lists forces and moments applied on joint elements.

SOLUTION: FE Forces
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

Elem

Element number or ALL (or blank). Lists joint element forces and moments on the specified element(s).

Notes

Valid for MPC184 joint elements. See FJ for information on specifying forces and moments.

Menu Paths

Utility Menu>List>Loads>Joint Element Forces>On Picked Element
