Chapter 1: About This Reference

Welcome to the Command Reference. This reference contains a complete dictionary of detailed command descriptions, arranged in alphabetical order. It is the definitive resource for correct command usage, providing comprehensive specifications for every argument of every command.

The Command Reference is intended to provide information about individual commands. Although the reference also contains lists of commands arranged by functional grouping, it is not intended to be your primary source of procedural information. For introductory information and procedural guidelines concerning when and where to use commands, see the appropriate analysis guide for your application.

A command mentioned in the various analysis guides implies a link to the detailed command description given in this reference. For ordering purposes, the alphabetical ordering of commands that begin with a star (*) or a slash (/) ignores those symbols.

A printed copy of the Commands Quick Reference Guide is available on the ANSYS customer site in the documentation area.
