/ERASE

/ERASE
Specifies that the screen is to be erased before each display.

GRAPHICS: Set Up
Valid Products: DesignSpace | Pro | Premium | Enterprise | PrepPost | Solver | AS add-on

Command Default

Previous /ERASE or /NOERASE setting will be used. Initially defaults to /ERASE setting.

Notes

Erase occurs with the next display request, but before the display is actually started. /NOERASE can be used to suppress the automatic screen erase.

This command is valid in any processor.

Menu Paths

Utility Menu>PlotCtrls>Erase Options>Erase Between Plots
