# Lib - 公共库

存放项目对ANSYS APDL编程的抽象和封装，提供核心功能和工具类，供其他模块调用。

## 结构

```
lib/
    ├── __init__.py       # 包初始化
    ├── built_in.py       # 内置类型抽象（如字符串、数字等）
    ├── files.py          # 文件操作封装
    ├── files_obj/        # 文件对象子包
    │   ├── __init__.py
    │   └── file.py       # 文件类封装
    ├── math.py           # 数学运算封装
    └── string.py         # 字符串处理封装
```

## 模块组织

新模块应按功能划分，一个模块一个职责。

| 模块 | 说明 |
|------|------|
| `built_in` | 内置类型（字符串、数字等）的抽象封装 |
| `files` | 文件操作相关功能 |
| `files_obj` | 文件对象封装 |
| `math` | 数学运算封装 |
| `string` | 字符串处理封装 |
